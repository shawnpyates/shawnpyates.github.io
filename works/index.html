Draft Machine

A web application that enables users to create and join custom drafts for rec or fantasy sports leagues which take place in real time.

MD Flashcards

For me, flashcards are an incredibly useful learning tool, but when it comes to learning new software technologies, I get very limited mileage out of currently existing flashcard apps due to their lack of support for Markdown code snippets. The backend for MD Flashcards is the first API I've written in Elixir/Phoenix, and the app has incidentally been a great help in bootstrapping my own Elixir learning journey.

React Timepicker Lite

A lightweight timepicker widget for React (published as an NPM package) that aims to prevent invalid input as early as possible. Triggers an onSuccess hook when unfocussed with valid input that provides an HH:mm timestring.
